A full-stack event sharing platform where users can:

- Create events
- Generate shareable links
- Collect RSVPs (Yes / No / Maybe)
- View attendee responses via secure host link

## Tech Stack

- Next.js (App Router)
- TypeScript
- Supabase (PostgreSQL + RLS)
- Tailwind CSS

## Features

- Dynamic event pages
- Secure host-only RSVP dashboard
- API routes with service role validation
- Row Level Security (RLS)
- Modern App Router architecture

## Future Improvements

- Email invites
- Authentication
- Custom questions
- RSVP editing
- Deployment to Vercel
